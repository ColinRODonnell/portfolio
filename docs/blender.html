DIGIT 220N: 2D & 3D Procedural Animation

This class is taught by Mr. Hartung in the Burke Building at Behrend. He also teaches ART168 and DIGIT 409. In this project we learned how to create

realistic motion in a complex animation software. I began this project by building racetrack pieces and extending them with different attributes within Blender.

Then, I created detailed mountains with geometry nodes as well as animating a snail, my subject, across the track.
